## Required submission evidence

- Working public project or unrestricted judge-accessible demo.
- English project story synchronized with actual verified behavior.
- Public YouTube demo no longer than three minutes with audible narration.
- Narration explains what was built, how Codex was used, and how GPT-5.6 was
  used.
- Repository includes setup, sample data, testing instructions, limitations,
  license, and Build Week scope.
- Primary Codex build thread `/feedback` Session ID.
- Public repository with relevant license, or private repository shared with
  the competition testing addresses.
- Third-party code, data, assets, and licenses documented.
- No copyrighted music or unlicensed third-party media in the video.
- Submission finalized before July 21, 2026 at 5:00 PM Pacific Time.

## Before final submission

- Replace future-tense claims in the project story with verified results.
- Add at least three clear 3:2 screenshots: zero state, learning ecosystem, and
  evidence/ablation result.
- Verify every metric shown in the video can be reproduced from a saved report.
- Test the live link from a private browser window with no existing session.
- Confirm the deployed commit SHA matches the repository release.
- Run a final secret scan of the repository and video frames.
- Generate and record the `/feedback` Session ID from the primary build thread.
